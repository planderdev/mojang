장지의 차이, 알고 계신가요?

  • 공설 묘지
    비용이 거의 발생치 않지만 현재 빈자리 없음 / 100 만원 이하
  • 사설 묘지
    봉분, 비석 설치 비용 발생 / 약 700 만원
  • 1차 장지 (화장장)
    고인 주소지 = 화장장 주소 (관내)
    고인 주소지 ≠ 화장장 주소 (관외)
    / 50 ~ 100 만원
  • 2차 장지 (수목장)
    자연으로 회귀 하는 섭리에 맞는 장법
    / 종류에 따라 비용 차이 발생
  • 2차 장지
    (봉안당=납골당)
    실내에 유골함에 안치 (유품 보관 가능)
    / 종류에 따라 비용 차이 발생
  • 2차 장지
    (봉안담, 봉안묘)
    야외에 설치한 시설에 안치 (유품 보관 불가)
    / 종류에 따라 비용 차이 발생
  • 2차 장지
    (산골, 바다장)
    자연에 뿌리는 방식 (산, 바다 등)
    / 종류에 따라 비용 차이 발생
